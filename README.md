# Keffect Marketplace (Initial Commit)

A comprehensive online marketplace architecture using **PHP Laravel** for both the customer-facing storefront and administrative operations, with **Linden Scripting Language (LSL)** integration for in-world interactive commerce and **Linden Dollar (L$)** payment authorization workflows.

## Objectives

- Deliver a user-friendly, secure, scalable marketplace platform.
- Use Laravel for:
  - Front-end UI (Blade + Livewire/Inertia patterns)
  - Back-end admin tooling (catalog, orders, user management, moderation)
- Integrate LSL endpoints for in-world vendors/kiosks.
- Prioritize **Linden Dollar payment preference**:
  - Webhook/event validation
  - Account debit after explicit in-world permission authorization
  - Reliable transaction state sync between in-world objects and Laravel backend

## Feature Set

### Customer-facing
- Product discovery and category navigation
- Product details, variants, inventory visibility
- User registration/login/profile management
- Seamless checkout and order history
- Wishlists
- Product reviews + moderation support
- Messaging/support threads

### Administrative
- Product CRUD with media and variant management
- Order processing and refund workflows
- User account moderation and role-based permissions
- Review moderation queue
- Support inbox / case handling
- Audit logs for sensitive actions

### Payment + LSL (L$ preference)
- In-world purchase intents generated by LSL object interactions
- Permission-based debit authorization flow in LSL (explicit user consent)
- Laravel webhook/event validation for payment confirmations
- Idempotent transaction records and replay-safe callback handling
- Delivery confirmation back to LSL vendors
